# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

The classification is based on flower measurements such as sepal length, sepal width, petal length, and petal width.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-Learn

## 🤖 Machine Learning Algorithm
- Random Forest Classifier

## 📊 Workflow
1. Load the Iris Dataset
2. Data Preprocessing
3. Split Data into Training and Testing Sets
4. Train the Random Forest Model
5. Predict Flower Species
6. Evaluate Model Accuracy

## 📈 Results
- Successfully classified Iris flower species.
- Achieved high prediction accuracy on test data.

## 📂 Dataset Features
- SepalLengthCm
- SepalWidthCm
- PetalLengthCm
- PetalWidthCm
- Species
